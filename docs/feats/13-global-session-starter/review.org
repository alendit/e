* Implementation Summary

- Added =e-chat-starter= as a separate presentation shell depending on the chat
  shell.
- Added =e-chat-start-here= to capture bounded source context from any buffer,
  create a real chat session, submit one prompt, and render the one-shot result
  in a compact =special-mode= popup buffer.
- Added starter popup actions: =c= continues the same session in the normal chat
  shell, =o= opens the latest answer in a separate buffer, =y= copies the answer,
  and =q= dismisses the popup interactively.
- Added =C-c C-c= in the starter popup as the chat-consistent shortcut for
  continuing/submitting the starter session.
- The starter popup now focuses its temporary window when opened, and action
  selection closes the temporary popup window before displaying the target chat
  or answer buffer.
- Added narrow public chat helpers for source capture, prompt reference
  formatting, session creation, session submission, and opening an existing chat
  session.
- Starter-created sessions are persisted with =:origin :global-session-starter=
  and source-reference metadata.
- Startup now requires and registers the =global-session-starter= shell manifest.
