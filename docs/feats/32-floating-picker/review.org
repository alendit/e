* Review loop

The first subagent review found three actionable issues:

- picker =:actions= were implemented as a dispatcher but action keys were not
  installed in the live picker keymap;
- terminal fallback completion used =:candidate-key= but did not expose the
  planned =:candidate-line= annotation metadata;
- posframe teardown only hid the child frame and did not delete it from a
  kill-buffer lifecycle path, and the new module was missing from
  =e-dev-reload=.

The fixes install per-spec action keys into the picker buffer's local map, add
fallback completion annotation metadata, delete the posframe from the picker
kill hook, and add =lisp/shells/e-picker.el= to the dev reload module list.
Regression coverage exercises each path through the actual keymap, completion
metadata, and kill hook behavior.
